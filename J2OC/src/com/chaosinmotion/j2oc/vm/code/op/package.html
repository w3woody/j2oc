An operator represents an interemediate operation in a collection of operations that comprise an instruction. An instruction represents a single generated statement.

The difference is deliberate: when one instruction evaluates a collection of operators, we can express this as a single C or Objective C statement: thus, the operators:

push 1
push 2
add
push 3
mul
arraynew int
will generate new[(1+2)*3];

This implies that if we execute a statement that doesn't return anything, then we need to close out our operators by using temporary registers.
